rw-sqlite3-encrypt
==========

# Description

rw-sqlite3-encrypt is the [go-sqlite3-encrypt](https://github.com/CovenantSQL/go-sqlite3-encrypt) custom version, where SQLITE_MAX_VARIABLE_NUMBER is raised to 32767.
